# Koobal Search Engine - v0.8.5.3

Predictive suggestions for the stock VAB/SPH parts search bar.
Type to get live matches for parts, stock filters (category, manufacturer,
diameter, tags, modules, resources, tech), and mods/authors - then click a
row to apply it. Recent searches appear when the field is empty.

Attaches a dropdown under the native search field. Does not replace the
stock parts UI.

## Install

1. Close KSP.
2. Copy `GameData/KoobalSearchEngine/` into your KSP `GameData/` folder
   (merge/overwrite).
3. Launch KSP and open the VAB or SPH.
4. Confirm in `KSP.log`: `[Koobal] Koobal Search Engine v0.8.5.3 active.`

## Requirements

- **KSP 1.12.x** (tested on 1.12.5)
- **Harmony 2** - `GameData/000_Harmony/` must be present
  https://github.com/KSPModdingLibs/HarmonyKSP/releases

Do **not** install alongside **Koobal Native Search** (or other mods that
replace/hijack the same stock search bar).
